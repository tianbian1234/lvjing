课程介绍

初级机器人

Primary Robot

初级机器人课程采取了欧美先进的项目制学习方式。学生不再是被动地接受知识灌输，而是以项目为基础进行学习，在项目进展的过程中思考、动手、实践、反思，主动去寻求方法和答案。每一位学生都会拥有自己的学习账号，这个账号将会在软件后台记录其学习数据，生成个人学习报告。 通过用户个人档案和个性化学习，与大数据相结合，用技术和智能为每一位学生量身打造适合自己的个性化学习方案。独有的3D建模部分，通过动态演示，辅助学生透彻了解机器人项目，使搭建过程更加简单明了，提高了学习的兴趣和效率。

中级机器人

intermediate Robot

以多元智能、STEM体系为教育理念，充分把握世界最新潮流，让孩子更具国际视野，囊括数学、人工智能、造型艺术、软件工程、计算机、机械工程、生物工程等七大学科知识交叉，孩子们在学习的过程之中，不但要对涉及到的知识熟练运用，还要把知识在实操过程中熟练的转化为技能，迅速进行外观设计，功能编程，线路搭建，场地判断、流程预设等多个环节；为培养未来开放型工程、IT、智能、机械等高精尖人才奠定基础； C++语言与图形化模块拖拽相结合，帮助孩子更好理解高级语言，更快入门。

高级机器人

Advanced Robot

智能核心组件、智能感应器组件、驱动器组件、机械结构、机械零配件。各组件与CastleRock™云学习平台相辅相成，从最核心处保证了课程与软硬件一体化。不同于一般的拖图编程，高级课程采用C++语言编程。通过技术手段，将复杂的编程学习简单化，避免了艰难晦涩的底层代码步骤，让学生专注于机器人的创意设计。轻松入门C++.

无人机课程

Advanced Robot

通过学习，强化手，脑协调能力，借助模拟器飞行与竞赛，充分调动孩子们的手/眼/脑的运作协调，在强烈的感官互动中发展他们的创造性潜能。培养孩子的创新精神、克服困难的能力、竞争意识和竞技精神、团队协作能力、锻炼孩子实践动手能力

航模课程

Advanced Robot

航模运动是一项在全国范围内深受青少年喜欢的运动。航模的制作和试飞，可以更好地培养学生细致入微的观察力，丰富他们的想象力；增强学生的动手操作能力，锻炼他们分析问题和解决问题的能力。不仅能为学生的知识应用和个人才能的展示与发展提供自由的活动空间，更能为现代科技的普及与应用创造条件。

3D打印

Advanced Robot

3D打印技术可以给学生的“学习方式”带来新的思考，让抽象的教学概念更加容易理解，可以激发学生对科学、数学尤其是工程和设计创意的兴趣，带来实践与理论、知识与思维、现实与未来三方面的相互结合。具体表现为：①3D打印让学生的想象更容易变成现实，培养学生的创新意识，鼓励学生的创新实践。②学习运用简易建模软件，发展学生立体空间思维。③通过3D打印实体的触觉过程，为学生建立一种新型的学习通道。④选择贴近生活的建模主题，培养学生解决生活实际问题的能力。

arduino创意课程

Advanced Robot

Arduino创意机器人遵循STEM科学、技术、工程、数学四位一体的理念，学生在运用Arduino平台进行机器人创造开发的时候，把学习到的零碎知识与机械工程转变成一个探究世界相互联系的不同侧面过程，使得学生在杂乱无章的学习情境中获得设计能力、合作能力、问题解决能力和实践创新能力的提升。

Scratch课程

Advanced Robot

Scratch以孩子喜欢的形式，结合图形化编程模块开始，逐步推进到专业的Python和C编程，我们将游戏和故事贯穿整个教学体系，让编程简单得就像搭积木。孩子能够通过动漫式、游戏化的课程任务，最专业的教师指导学习编程，训练逻辑思维与想象力，提高创造力与动手能力。鼓励学生共同完成编程作品，引导孩子合作与竞争，学会分享，应对困难，培养学生领导能力与协作能力。

新能源课程

Advanced Robot

新能源课程是集各种新能源技术、环境保护和能源危机教育于一体的综合性课程。在学习的同时，对学生进行环境保护、能源危机的教育，传播科学的发展理论，增强学生的保护、节能和利用新能源的意识；将数学、物质、科学、物理学、和工程学相结合，开阔学生的视野，激发学生学习的兴趣。

高级创作课程

Advanced Robot

学习高级编程语言、机械原理及设计及电路设计 。学生实现自己的创意，通过电路设计，机械设计，结合3D打印等技术，编写高级语言，如c++等，完成自己理想的作品。
